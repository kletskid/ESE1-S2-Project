# emios_frequency_modulated_pwm

## Overview

The eMIOS project is a simple demonstration example of the SDK eMIOS driver to use eMIOS as PWM generator.  
It configures eMIOS UC (Unified channel) as Output Pulse Width and Frequency Modulation Buffered (OPWFMB) mode.  
Every two seconds, eMIOS will update PWM frequency from 10kHz to 20kHz, or from 20kHz to 10kHz.  
eMIOS generate 10kHz PWM at first.
